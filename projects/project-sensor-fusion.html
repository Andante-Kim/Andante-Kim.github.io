Sensor Fusion Framework

Modular multi-sensor perception system

Python C++ ROS2 Kalman Filter

üéØ Project Overview

Designed and implemented a modular sensor fusion framework that intelligently combines data from cameras, LiDAR, and radar sensors to create a unified, robust perception of the environment. The framework uses Extended Kalman Filtering (EKF) and particle filtering techniques to fuse heterogeneous sensor data while accounting for their different characteristics and uncertainties.

This framework serves as the foundation for autonomous vehicle perception stacks, providing reliable object detection, tracking, and state estimation even when individual sensors fail or provide noisy measurements.
